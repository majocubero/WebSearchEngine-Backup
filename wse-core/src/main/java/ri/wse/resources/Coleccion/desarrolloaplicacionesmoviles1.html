Aprende a Programar Aplicaciones para Android

Escrito por KhronoX el 2 julio 2011 a las 12:15

Android es actualmente la plataforma con mayor cuota de mercado en el mundo (según reportes de la IDC) superando a Nokia, iPhone y Blackberry. Se estima que 300,000 teléfonos móviles con Android son activados diariamente. La gran demanda por dispositivos móviles que corran bajo esta plataforma, genera un gran interés por parte de los desarrolladores que desean crear aplicaciones para un mercado en pleno auge. Si eres desarrollador/a y deseas aprender a crear aplicaciones para Android, aquí encontrarás una serie de recursos que te ayudarán a iniciarte en esta gran aventura.

Para desarrollar aplicaciones para dispositivos móviles que funcionen bajo el sistema operativo Android, deberás tener nociones básicas sobre programación. Si conoces el lenguaje C o C++, entonces no tendrás muchos problemas, ya que para crear aplicaciones que funcionen en Android vas a tener que programar en Java cuya sintaxis es casi la misma. Lo que sí vas a tener que aprender es a utilizar una serie de librerías y componentes propios de la plataforma. Pero no te preocupes que todos estos detalles los encontrarás en los recursos que verás a continuación.

Guías, Tutoriales y Recursos para Aprender a Desarrollar Aplicaciones para Android

Lo primero que debes aprender como desarrollador de aplicaciones Android es el lenguaje de programación Java. No es necesario ser un experto en el manejo del lenguaje Java, pero al menos debes conocer y manejar con fluidez el lenguaje.

Aprende el Lenguaje de Programación Java

  • Manual de Java en pdf: Aprenda Java como si estuviera en primero. El manual es poco antiguo, pero lo importante es aprender a manejar el lenguaje Java, su sintaxis, sentencias, instrucciones básicas.
  • Manual de Java online: Manual bastante completo y detallado

Aprende a Desarrollar Aplicaciones para Android

  • Desarrollo en Android sgoliver.net: Completa web en donde encontrarás una gran cantidad de información para aprender a desarrollar aplicaciones para Android. Se tratan temas como conceptos básicos, interfaz de usuario, menús, widgets, bases de datos, mapas y más.
  • Guia Oficial de Android para desarrolladores (inglés)
  • Desarrollo de aplicaciones para Android Gingerbread y Eclipse (inglés)
  • Tutoriales de DroidDraw (inglés). Esta aplicación te permitirá diseñar la interfaz de usuario de tus aplicaciones Android de manera intuitiva.
  • Google Code University (inglés). Videos, cursos y lecturas sobre desarrollo de aplicaciones en Android.
  • Curso Android de 10 Capítulos. Este curso está en castellano y puedes verlo online o descargarlo en pdf. Además encontrarás recursos adicionales

Herramientas Necesarias para Desarrollar Aplicaciones para Android

  • Para desarrollar aplicaciones en Android necesitas una computadora con sistema operativo Windows, Max OS X (intel) o Linux (i386)
  • Descarga el Android SDK compatible con tu sistema operativo. Puedes descargarlo en este enlace.
  • Un entorno de desarrollo integrado gratuito como Eclipse para desarrolladores Java.
  • DroidDraw es un constructor/diseñador/editor de interfaz de usuario tipo WYSIWYG para aplicaciones tablet y de celular en plataforma Android.

 

Espero que estos recursos te sean de utilidad y puedas desarrollar tus aplicaciones para Android.
