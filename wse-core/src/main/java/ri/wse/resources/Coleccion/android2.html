APN Backup & Restore

19. marzo 2010
michel4 comentarios
(Sin Votos)
 Loading ...

Algunas aplicaciones conjugan sencillez y efectividad y una de ellas es, sin lugar a dudas, APN Backup & Restore.

APN, acrónimo de Access Point Name (nombre del punto de acceso), define la configuración que permite conectarse a Internet mediante una conexión móvil y varía en función del operador y el país. Es por lo tanto muy práctico salvaguardar aquellos APN que usamos frecuentemente para no tener que introducirlos manualmente.

APN Backup & Restore permite tanto hacer una copia de seguridad de dichos APN como recuperarlos en cualquier momento evitándonos la tediosa labor de hacerlo manualmente.
