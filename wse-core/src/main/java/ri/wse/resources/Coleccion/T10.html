    Herramientas para desarrollo Front-end

    Por Sergio | Creatividad, Pgina Web, Web | 0 comentarios | 14 noviembre, 2016 | 0

    En estas pocas, se habla mucho de la interaccin y el diseo de los sitios web y las aplicaciones mviles, ya que son herramientas de trabajo o entretenimiento que son diariamente usadas por miles y millones de personas. Es por eso que las empresas, grupos y dems personas que se dedican al desarrollo de aplicaciones y productos de software deben considerar siempre la interaccin y la experiencia que los usuarios tienen con respecto al software de uso diario.

    Ejemplos de estos son:

    Material Design

    Es un patrn de diseo que la empresa Google ha creado para ser implementado en aplicaciones mviles. Este patrn establece varios lineamientos que deben seguirse en cuestin de posicin, tamao, formas, efectos y patrones para que la interaccin sea ms intuitiva y sencilla. Google ha creado de igual manera un framework para diseo web que sigue al patrn de material design, llamado materialize. Puedes encontrar un framework basado en Material Design en Materialize.

    Bootstrap

    Bootstrap es un framework de diseo web que obedece diversos patrones dirigidos a proyectos que requieren de alta adaptacin a los distintos tipos y tamaos de pantalla, tanto de escritorio como mviles. Bootstrap es uno de los frameworks ms utilizados actualmente para el diseo y desarrollo de interfaces. Puedes encontrarlo en Bootstrap.

    Office UI Fabric

    Es un framework de diseo web que obedece a los patrones de office. Es relativamente nuevo comparado con otros frameworks. Los elementos que contiene para el desarrollo sirven para crear aplicaciones que requieren de un lapso prolongado de escritura, ya que es muy familiar (literalmente) a las aplicaciones de la clsica paquetera. Puedes encontrarlo en Office UI Fabric.
