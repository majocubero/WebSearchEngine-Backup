Casi en la cuenta atrás para Navidad os traemos esta aplicación para Android gratis que ha sido un éxito en el Android Market.

Algunos de los ajustes disponibles en Christmas Tree Live Wallpaper:

- Ajusta el sonido con un solo tap en la pantalla.

Como de costumbre este Live Wallpaper es totalmente gratuito y compatible con la mayoría de dispositivos Android en el mercado.

FBReader

Lector para ebooks o libros electrónicos con el que podrás visualizar archivos con extensión epub, oeb y fb2 (.zip).

Esta aplicación Android es gratis y te permitirá leer cómodamente desde tu dispositivo Android los libros y documentos favoritos.

Algunas de las características de FBReader:
- Rápido y fácil de usar.
- Personalizable.
- Fuente de letra True Type / Open Type.
- Navegador incluido.
- Hasta 14 idiomas para seleccionar.

Ademas FBReader cuenta tiene integración para diccionarios externos como FreeDictionary.com, ColorDict o Fora Dictionary.

      Google Authenticator

      Google te presenta esta aplicación gratis para dispositivos Android, con la que podrás mejorar la seguridad de tu cuenta Google, para productos como Gmail, Google Analitics, etc.

      Google Authenticator genera códigos de verificación, para proteger tu identidad en lo que comúnmente se denomina verificación en 2 pasos (2-step verification).

      Instalando esta aplicación obtendrás seguridad extra para tu cuenta de Google ya que además de tu contraseña ordinaria, tendrás que introducir el código único generado por tu teléfono.

      - Instalación automática gracias a códigos QR.
      - Soporte para múltiples cuentas.
      - Disponible en varios idiomas.
