Comenzar a desarrollar aplicaciones para Android con Java y Eclipse

Android… es una marca, es un TopOfMind, denominado como “El Windows para Dispositivos Móviles” es la nueva tendencia en el mercado tecnológico y está marcando una pauta impresionante. Muchos desarrolladores están emprendiendo en comenzar a desarrollar aplicaciones para este sistema operativo. Pero… ¿Como comenzar? … A continuación una sencilla guía de los elementos e instalaciones necesarias para que comiences a verte inmerso en este nuevo y maravilloso mundo tecnológico “móvil” con Android.

Para comenzar, existen ciertos requerimientos básicos que debes cumplir para comenzar la instalación:

Requerimientos

Sistema Operativo

  • Windows XP (x86)
  • Windows Vista
  • Windows 7
  • Mac OS X 10.5.8 o superior (x86)
  • Ubuntu Lucid Lynx o superior (x86)*

* Para usarlo con Ubuntu debes tener previamente instalado GNU C Library (glibc) 2.7 o superior.

Entornos de Desarrollo *

  • Eclipse 3.5 (Galileo) o superior [DESCARGAR AQUÍ]
  • NetBeans 6.8 y 6.9 [DESCARGAR AQUÍ]
  • IntelliJ IDEA [DESCARGAR AQUÍ]

* En el presente tutorial explicaré el procedimiento solo con Eclipse.

Development Kits

  • JDK 6 (Java Development Kit) * [DESCARGAR AQUÍ]

* Java 7 no funciona aún con Android (aunque puede tenerse instalado en paralelo con Java 6).

Requerimientos de Hardware

  • Almacenamiento en disco por cada componente a instalar:*
Componente Tamaño aproximado Comentarios
SDK Tools 35 MB Requerido
SDK Platform-tools 6 MB Requerido
Android Platforms 150 MB (c/u) Es requerido al menos una plataforma
SDK Add-on 100 MB (c/u) Opcional
USB Driver for Windows 10 MB Opcional. Solo para Windows.
Ejemplos (por plataforma) 10 MB Opcional
Documentación Offline 250 MB Opcional

* No te que el espacio en disco especificado no incluye al entorno de  desarrollo ni el kit de desarrollo, los cuales debe tener previamente instalados para comenzar el procedimiento.

Si cumples los procedimientos y has instalado previamente los elementos necesarios, podemos proceder a la instalación principal.

Instalar y preparar entorno para comenzar a desarrollar en Android

1. Descargar en SDK de Android. [DESCARGAR AQUÍ] (descargar el que corresponda a tu sistema operativo).

2. Instalar el SDK previamente descargado (ejecutar como administrador en Windows 7).

3. Ejecutar el SDK Manager tras la instalación, e instalar los “platforms” correspondientes a las versiones de Android en las que piensas desarrollar.

4. Instalar el Plug-In ADT para Eclipse (en caso de que vayas a trabajar con Eclipse). A continuación los pasos para instalarlo.

4.1. Primero que todo se debe tener instalado Eclipse 3.5 Galileo o superior (Indigo). Se recomienda utilizar la versión Eclipse Classic.

4.2. Iniciar Eclipse

4.3 Help –> Install New Software…

4.4 Clic en “Add” en la esquina superior derecha

4.5 En la ventana “Add Repository” escriba “ADT Plugin” como nombre y la siguiente URL como locación:

https://dl-ssl.google.com/android/eclipse/

4.6 Haz Clic en “Ok”

NOTA: Si tienes problemas intenta colocando en la URL de la locación “http” en vez de “https”.

4.7 El repositorio se cargará y aparecerá un elemento con una casilla de verificación al lado. Clic en la casilla de verificación para activarla.

4.8 Clic en “Next”

4.9 En la siguiente ventana veras una lista de las herramientas que serán instaladas. Clic en “Next”

4.10 “Lea” y acepte los Acuerdos de Licencia o License Agreements. Clic en “Finish”

NOTA: Si te aparece una advertencia de seguridad, Clic en “Ok”

4.11 Cuando las instalación se complete. Reinicie Eclipse.

4.12 Cuando se inicie Eclipse, se iniciará un asistente pidiendo que instalemos el SDK de Android con sus respectivos Platforms. Como éste paso ya lo ejecutamos previamente, seleccionamos la segunda opción y escribimos la ruta donde instalamos nuestro Android SDK. Se instala por defecto en “C:\Archivos de programa\Android\android-sdk” o “%PROGRAMFILES%\Android\android-sdk”

Ya podemos comenzar a desarrollar para Android directamente en Eclipse.
